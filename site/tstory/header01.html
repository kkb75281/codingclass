사이트 header 유형 #1

사이트 상단에 위치한 header 부분을 만들어 봅시다. 오늘은 간단해서 다들 성공하실거에요 ☺️

1. CSS 살펴보기

/* reset */
em, i, address {
    font-style: normal;
}

+ reset에 폰트 스타일 초기화 추가 

/* headerType */
.header__inner {
    width: 100%;
    height: 70px;
    display: flex;
    align-items: center;
    justify-content: space-between;

    font-size: 30px;
    font-weight: 700;
    text-transform: uppercase;
}
.header__logo em {
    font-size: 18px;
    font-weight: 400;
}
.header__menu {
    width: 60%;
    text-align: center;
}

}

}

}
.header__member {
    width: 20%;
    text-align: right;
}
.header__member a {
    font-size: 16px;

    border-radius: 50px;
    transition: all 0.3s;
}
.header__member a:hover {
    background-color: #000;
    color: #fff;
}
