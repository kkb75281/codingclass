1부터 100까지 출력하기

1.1 while문을 이용해서 1부터 100까지 출력하시오 !

let x = 100;
    let y = 200;                
    let z = "javascript";  
    
    x = 300;                // 변수 x의 값이 100에서 300으로 변경됨
    y = 400;                // 변수 y의 값이 200에서 400으로 변경됨
    z = "jquery"            // 변수 z의 값이 "javascript"에서 "jquery"로 변경됨
    
    document.write(x);
    document.write(y);
    document.write(z);
결과보기
300
400
jquery
