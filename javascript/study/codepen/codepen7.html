글자가 '점핑저점핑'하는 효과를 만들어봅시다 !

#1. HTML 코드를 확인해보자

글자마다 span으로 감싸서 만들어줍니다.

<h1>
    <span>I</span>
    <span>'</span>
    <span>m</span>
    <span>G</span>
    <span>y</span>
    <span>u</span>
    <span>p</span>
    <span>p</span>
</h1>

#2. CSS(SCSS) 코드를 확인해보자

    width: 100%;
    height: 100%;

    display: flex;
    justify-content: center;
    align-items: center;

    height: 100px;
}
