🤓 jQuery, 사용해볼까?

jQuery를 사용하기 위해서는 jQuert 라이브러리를 웹페이지에 연결해서 사용해야 합니다.
각 버전별 브라우저 호환 여부가 다르기 때문에 확인하시고 연결하시면 됩니다 '3'


📌 직접 다운로드 방식

1) jQuery 라이브러리 다운로드
공식 홈페이지 내 Download jQuery v3.6.1 을 다운로드 받아 사용합니다.

2) HTML 문서에서 jQuery 라이브러리 연결
<script src="jquery-3.6.1.min.js"></script>
html 문서를 jQuery 라이브러리와 같은 경로에 저장합니다.

📌 CDN 방식

1) CDN 복사하기
공식 홈페이지 내 jQuery CDN을 복사합니다. 그 밖의 Google / Microsoft / CDNJS / jsDelivr CDN도 사용하실 수 있습니다.

2) HTML 문서에서 jQuery 라이브러리 연결
<script src="jquery-3.6.1.min.js"></script>
html 문서를 jQuery 라이브러리와 같은 경로에 저장합니다.

🤨 jQuery 기본 형식

<script src="jquery-3.6.0.min.js"></script>
<script>
    $(document).ready(function(){
        실행문;
    });
</script>

jQuery의 기본 형식은 $(document).ready( ) 입니다.
ready( )는 jQuery 이벤트로 자바스크립트의 load 아밴트와 유사한 기능을 가지고 있습니다. 

<script src="jquery-3.6.0.min.js"></script>
<script>
    $(function(){
        실행문;
    });
</script>

document.ready를 생략하여 표현할 수도 있습니다.
