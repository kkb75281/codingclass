charAt( )에 대해 알아보자 !

String.prototype.charAt( );
charAt( ) 메서드는 문자열에서 특정 인덱스에 위치하는 문자를 반환합니다.

construction(구문)

str.charAt("index")

index(순서)

0과 문자열의 길이 - 1 사이의 정수값이 올 수 있습니다.
인덱스 값을 쓰지 않는다면 기본값으로 0이 설정되고 첫 문자를 반환합니다.
만약 인덱스가 문자열 길이보다 큰 경우 빈 문자열 " " 을 반홥합니다.

# 예제를 통해 익혀봅시다.

const str1 = "javascript reference";

const currentstr1 = str1.charAt();
const currentstr2 = str1.charAt("0");
const currentstr3 = str1.charAt("1");
const currentstr4 = str1.charAt("2");
