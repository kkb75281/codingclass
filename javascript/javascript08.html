  • 01. 숫자 메서드 : Number.MAX_VALUE : JavaScript에서 표현할 수 있는 최대 숫자 값 ≒ 1.7976931348623157E+308
  • 02. 숫자 메서드 : Number.MIN_VALUE : JavaScript에서 표현할 수 있는 가장 작은 양수 값 ≒ 0에 가장 가까운 숫자
  • 03. 숫자 메서드 : Number.MAX_SAFE_INTEGER : JavaScript에서 최대 안전 정수 ≒ 9007199254740991
  • 04. 숫자 메서드 : Number.MIN_SAFE_INTEGER : JavaScript에서 안전한 최소 정수값 ≒ -9007199254740991
  • 05. 숫자 메서드 : Number.Nan : Not-A-Number 값을 나타냄 (Nan ≒ 숫자로서의 정상적인 값이 아님)
  • 06. 숫자 메서드 : Number.isFinite : 주어진 값이 유한 숫자인 경우 true, 그렇지 않으면 false 반환
  • 07. 숫자 메서드 : Number.isInteger : 주어진 값이 정수인 경우 true, 그렇지 않으면 false 반환
  • 08. 숫자 메서드 : Number.isNaN : 주어진 값이 있는 숫자인 경우 true, 그렇지 않으면 false 반환
  • 09. 숫자 메서드 : Number.prototype.toFixed : 숫자를 고정 소수점 표기법(fixed-point notation)으로 표시
  • 10. 숫자 메서드 : Number.prototype.toPrecision : 지정한 정밀도만큼 반환
  • 11. 숫자 메서드 : Number.prototype.toString : 지정된 숫자 값을 나타내는 문자열을 반환
